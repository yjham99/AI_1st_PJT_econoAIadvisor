# 🪟 [알파 HQ] 윈도우 자율 가동(스케줄러) 등록 가이드

지휘관님, 이제 PC를 켜는 순간 별도의 직접 실행 없이도 **[알파 HQ]**가 백그라운드에서 자동으로 가동됩니다. 아래 절차를 1회만 수행해 주십시오.

---

### 1단계: 윈도우 작업 스케줄러 실행
1. 윈도우 검색창에 **"작업 스케줄러"** (Task Scheduler) 입력 후 실행.
2. 우측 [동작] 패널에서 **[기본 작업 만들기...]** 클릭.

### 2단계: 작업 설정
1. **이름**: `Alpha_HQ_Autonomous_Start` 입력 후 [다음].
2. **트리거**: **[컴퓨터 시작 시]** 또는 **[로그온할 때]** 선택 후 [다음].
3. **동작**: **[프로그램 시작]** 선택 후 [다음].

### 3단계: 프로그램 연결
1. **프로그램/스크립트**: [찾아보기] 클릭 후 아래 경로의 배치 파일 선택.
   - `c:\AI_Study_Beginer\1st_PJT_econoAIadvisor\start_sentinel.bat`
2. **시작 위치 (선택 사항)**: 반드시 아래 경로를 복사하여 입력해 주십시오. (매우 중요)
   - `c:\AI_Study_Beginer\1st_PJT_econoAIadvisor`
3. [다음] 클릭 후 [마침].

---

### 💡 운영 팁
*   **수동 가동**: 바탕화면에 `start_sentinel.bat` 파일의 [바로가기]를 만들어 두시면, 원클릭으로 모든 시스템을 재부팅할 수 있습니다.
*   **확인 방식**: 텔레그램에서 `/list` 명령을 보냈을 때 즉시 응답이 오면 시스템이 살아있는 것입니다.
*   **장애 발생 시**: 실행 중인 검은색 터미널 창들을 모두 닫고 `start_sentinel.bat`을 다시 실행하십시오.
